How to use DocsTemplate?

git clone https://github.com/Fire-The-Fox/DocsTemplate.git
mv DocsTemplate {name}
cd {name}

What next?

After you have successfully downloaded the repository, you can now customize the template.

Default file structure

./css
./css/framework.css
./css/index.css
./js
./js/framework.js
./index.html
./template.html

Every css file should be stored in the `css` folder and javascript files should be put inside of js folder

When you want to create a new page, copy template.html file and give it a specific name of the webpage, template.html is basicaly a blank website with defaults

How to add things into sidebar?

When you want to split one entire page into smaller sections and make programmer go to that specific section, they just can click on the name of section and website will send them to there

<a href="#id">{sectionName}</a>

#id reffers to an id, which you add to the specific element of a part of documentation and {sectionName} to the name of that section you want to be

When you want to add a link to another page of a documentation into the sidebar

<a class="sidebar-link" href="{file.html}">{pageName}</a>

{file.hmtl} represents name of a file of that page and to set name of a page in the sidebar, replace {pageName} with coresponding name
